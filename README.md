# 🚀 SEO Audit API (FastAPI)

A simple yet powerful SEO Audit API built using Python and FastAPI.  
This project analyzes basic SEO elements of a website and helps identify optimization opportunities.

---

## 📌 Features

- Extract website title
- Check meta description
- Identify H1 tags
- Basic SEO analysis
- Fast and lightweight API
- Interactive API documentation

---

## 🛠 Tech Stack

- Python
- FastAPI
- Uvicorn
